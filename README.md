# mini-adb-tools

一个简单的批量adb操作工具，学习项目，由Electron-Vite构建。

## 支持功能

- 批量连接
- 批量安装文件
- 批量启动远程控制

## Project Setup

### Install

```bash
$ pnpm install
```

### Development

```bash
$ pnpm dev
```

### Build

```bash
# For windows
$ pnpm build:win

# For macOS
$ pnpm build:mac

# For Linux
$ pnpm build:linux
```
